# 🥗 CalorieTracker

Веб-приложение для подсчёта суточной нормы калорий и контроля питания в течение дня.

---

## О проекте

CalorieTracker — это лёгкий инструмент без лишних функций и обязательной регистрации. Открыл, ввёл свои параметры, получил норму калорий и начал вести дневник питания. История сохраняется в браузере и не теряется при перезагрузке.

Для кого: люди, которые следят за питанием — начинающие в спорте, желающие похудеть или набрать массу, а также все кому важно понимать свою норму калорий без сложных приложений.

---

## Функциональность

- Расчёт суточной нормы калорий по формуле Миффлина–Сан Жеора
- Учёт веса, роста, возраста, пола и уровня активности
- Выбор цели: похудеть / удержать вес / набрать массу
- Дневник питания с разбивкой по приёмам (завтрак, обед, ужин, перекус)
- Прогресс-бар с отображением съеденных и оставшихся калорий
- Учёт белков, жиров и углеводов (БЖУ)
- История питания по дням с графиком за неделю
- Хранение данных в браузере (IndexedDB) — без потери при перезагрузке
- Адаптивный дизайн для компьютера и телефона

---

## Технологии

- HTML5 — разметка страниц
- CSS3 — стилизация и адаптивная вёрстка
- **JavaScript  — расчёт нормы, обработка форм, обновление прогресс-бара
- IndexedDB — браузерная база данных для хранения истории питания

---

## Запуск

Клонировать репозиторий и открыть home.html в браузере — никаких зависимостей и серверов не нужно.

```bash
git clone https://github.com/mesrop2008/calorie-tracker.git
cd calorie-tracker
```


Затем открыть файл home.html в любом браузере.
